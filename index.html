The Linux Foundation

Linux Foundation (LF) je neprofitni tehnološki konzorcij, ustanovljen leta 2000 kot združitev med Open Source Development Labs in skupino Free Standards za standardizacijo Linuxa, podporo njegovi rasti in spodbujanje njegovega komercialnega sprejetja. Poleg tega gosti in spodbuja sodelovalni razvoj odprtokodnih programskih projektov. Je glavna sila pri spodbujanju raznolikosti in vključevanja tako v Linux kot v širši odprtokodni programski skupnosti.

Zgodovina

Fundacija se je začela izvajati leta 2000, v okviru Open Source Development Labs (OSDL) in postala organizacija je danes, ko osdl združili s Skupino prostih standardov (FSG). Fundacija Linux sponzorira delo Linux ustvarjalca Linusa Torvaldsa in vodilnega vzdrževalca Grega Kroah-Hartmana. Poleg tega jo podpirajo člani, kot so AT&T, Cisco, Fujitsu, Google, Hitachi, Huawei, IBM, Intel, Meta, Microsoft, NEC, Oracle, Orange S.A., Qualcomm, Samsung, Tencent in VMware ter razvijalci iz celega sveta. V zadnjih letih je Fundacija Linux razširila svoje podporne programe z dogodki, usposabljanjem in certificiranjem, pa tudi odprtokodnimi projekti. Projekti, ki jih gosti fundacija Linux, vključujejo projekt linux jedra, Kubernetes, Automotive Grade Linux, ONAP (Platforma za avtomatizacijo odprtega omrežja), Hyperledger, Cloud Native Computing Foundation, Cloud Foundry Foundation, Xen projekt in mnoge druge.
